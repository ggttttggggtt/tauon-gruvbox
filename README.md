### Installation


Install any custom theme by placing the `.ttheme` in Tauon's theme folder

`~/.var/app/com.github.taiko2k.tauonmb/data/TauonMusicBox` (for flatpak)

`~/.local/share/TauonMusicBox/theme/` (for aur)
